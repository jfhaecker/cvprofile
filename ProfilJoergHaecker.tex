\section{Über mich}
\cvline{}{
Kundenbedürfnisse befriedigen - Das ist das Ziel. 
}
\cvline{}{
Funktionierende Software möglichst früh und  kontinuierlich ausliefern - Das ist der Weg.
}
\cvline{} { 
Ich bin bekennender ``Techi`` und habe langjährige Erfahrung als Führungskraft in der Softwareentwicklung.
Die Arbeitsweise von Entwicklungsteams in einem skalierten Umfeld ist mir somit bestens bekannt.  
Als Scrum Master lege ich besonders viel Wert auf die Erstellung eines ``Done``-Inkrementes, das auch wirklich ``Done`` ist. Im skalierten Scrum-Umfeld müssen dazu die Abhängigkeiten zwischen den Teams möglichst früh betrachtet werden.  
}
